shhan은 assembler로 제작된 간단한 한글 library입니다.
대충 만들었지만 다음과 같은 기본적인 기능이 제공됩니다.

1.한글의 입력 및 출력
2.허큘리스,EGA,VGA 지원
3.명조체 font하나만 있음
4.점단위의 위치선정이 가능
5.배경색 전경색 개별 지정 가능
6.software적으로 입력 mode변환 가능
7.가진 재주가 없다보니 크기는 초소형
8.공짜

이 라이브러리는 제가 직접 만든 겁니다.
코드는 물론 font까지 모눈종이에 대고 직접 그렸기때문에
모양이 솔직히 별로 볼품이 없읍니다.
더구나 벌레까지 생생하게 살아 숨쉬고 있을지도 모르죠.
현재까지는 특별나게 bug가 발견되지 않았습니다만 아마
있기는 있을 겁니다.
이렇게 불완전한 library를 upload하는 이유는 소스를 공개
하여 더 좋은 library를 만들고자 하는 분께 조금이나마
참고가 될까 해서입니다.
예제 testsh.exe를 실행시켜보면 기본적인 소개와 함수에
대해 설명을 합니다.
한가지 주의할 것은 이 library는 large memory model에서
만 동작을 합니다.
어셈블러 source는 MASM으로 compile을 하시고 C source
는 Turbo-C 2.0이나 BC 3.1중 하나를 사용하십시요.
허큘리스는 640*400및 720*348모드 두가지를 모두 지원하는
640*400모드를 사용할 경우는 gm을 1로 하고 720*348 모드
를 사용할 경우는 gm을 0으로 하여 shset함수를 호출하여야
합니다.
